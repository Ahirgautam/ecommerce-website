# ShopEase - A Ecommerce website
# A Spring Boot-based e-commerce web application.

## I have solo created this website as my major project of bca final year


## Technologies
- Backend
   - java Spring
   - Mysql Database
- Fronted
  - HTML
  - CSS
  - JS     

## Features
- User authentication using OAuth and normal(email-password)
- Product search, filter and cart management
- Admin dashboard
- Product order and tracking
